# AquaSoft Website

Official website for [AquaSoft](https://www.aquasoft-x.com) — AI Tools for Everyone.

## Tech Stack

- [Next.js](https://nextjs.org/) 16 (App Router, static export)
- [React](https://react.dev/) 19
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) 4
- [Framer Motion](https://www.framer.com/motion/)
- Deployed on [GitHub Pages](https://pages.github.com/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

Generates a static export to the `out/` directory.

## Pages

- `/` — Under construction landing page with interactive particle network
- `/luckydraw` — [LuckyDraw](https://www.aquasoft-x.com/luckydraw) product page (Microsoft Teams Bot)
